# Parking Management System (Java)

## 📌 Project Objective
This project is a Parking Management System developed in Java. The system collects information about the user, their vehicle, and the duration of parking. Based on this data, it calculates the parking fee and generates a receipt.

## 🚀 Features
- Collect user and vehicle details
- Parking duration input
- Automatic price calculation
- Receipt generation

## 🛠️ Tech Used
- Java
- Object-Oriented Programming (OOP)

## ▶️ How to Run
javac Main.java
java Main
